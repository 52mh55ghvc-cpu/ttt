# 情侣双人心情记录 - 使用说明

## 快速开始

1. 用浏览器直接打开 `index.html` 文件（双击或拖入浏览器窗口即可）。
2. 无需安装、无需联网，所有数据保存在本机浏览器的 LocalStorage 中。
3. 建议情侣共用同一台设备/同一浏览器使用，以便申请查看功能正常运作。

## 功能说明

### 0. 密码验证与页面分层（启动首页）

软件启动后**固定显示第 1 页「密码验证页」**，男女专属界面默认全部锁定隐藏，无法跳过。

| 页码 | 页面 | 说明 |
|------|------|------|
| 第 1 页 | 密码验证页 | 启动默认页，设置/修改/校验密码 |
| 第 2 页 | 男生界面 | 输入正确男生密码后进入 |
| 第 3 页 | 女生界面 | 输入正确女生密码后进入 |

**密码规则：**

- 男生、女生各有一套**独立专属密码**，互不通用
- 首次使用需在密码验证页分别「设置男生密码」「设置女生密码」（至少 4 位）
- 修改密码需先核验**原密码**，再录入新密码并确认
- 密码错误时停留在密码页并提示错误，不会跳转
- 刷新页面或重新打开软件后，自动回到密码验证页，需重新解锁
- 解锁后点击「🔒 锁定退出」可手动返回密码页

### 1. 双板块独立记录

页面分为 **男生** 和 **女生** 两大独立板块，各自记录自己的心情与日记，互不影响。

### 2. 365天固定周期

- 首次打开页面时，以当天作为记录周期第 1 天（共 365 天）。
- 每人每天只能保存 **一条** 记录；重复点击「保存今日记录」会覆盖当天已有内容。
- 周期结束后（第 366 天起）将无法继续记录。

### 3. 六种心情选择

每日必须从以下六种心情中选择一种：

| 心情 | 说明 |
|------|------|
| 极度开心 | 非常开心 |
| 轻微开心 | 有点开心 |
| 平淡普通 | 没什么特别 |
| 轻微不开心 | 有点低落 |
| 极度难过 | 非常难过 |
| 生气烦躁 | 生气或烦躁 |

### 4. 私密日记

- 每人可填写当日私密日记，**默认对方完全不可见**。
- 日记内容仅自己可在「365天记录列表」的「我的记录」栏中查看。

### 5. 申请查看对方今日日记

这是唯一的「解锁」途径，规则如下：

| 规则 | 说明 |
|------|------|
| 仅限当日 | 只能申请查看对方 **今天** 的日记 |
| 历史不可申请 | 过去日期的日记无法申请查看 |
| 需对方同意 | 发起申请后，对方会收到弹窗，可选择「同意」或「拒绝」 |
| 仅当日有效 | 同意后当天可查看；**次日自动失效，需重新申请** |
| 列表始终锁定 | 365天列表中，对方日记栏始终显示 🔒，不会展示任何内容 |

**操作流程：**

1. 在己方板块点击「申请查看」按钮。
2. 对方板块内会弹出确认窗口（仅对方可见操作按钮）。
3. 对方点击「同意」后，申请方可在「查看对方今日日记」区域看到内容。
4. 对方点击「拒绝」则无法查看。

### 6. 365天记录列表

每个板块底部有完整的 365 天日期列表：

- **我的记录**：展示自己当天的心情标签和日记全文。
- **对方（锁定）**：始终显示 🔒 已锁定，不展示任何对方内容。

列表会自动滚动定位到「今天」所在行（高亮显示 D 编号）。

## 数据存储

- 数据保存在浏览器 LocalStorage，键名为 `coupleMoodDiary_v1`。
- 清除浏览器缓存/网站数据会导致记录丢失，请注意备份。
- 更换浏览器或设备后数据不会同步。

## 注意事项

- 本应用为纯前端实现，无后端服务器，适合本地情侣共同使用。
- 申请查看功能：对方需用自己的密码解锁进入后，才能处理你的查看申请。
- 跨天后系统自动重置解锁状态与待处理申请，无需手动操作。
