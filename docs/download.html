情侣心情记录 · 下载

请用 Safari 浏览器 打开（不要用微信里点开）

方式一：直接打开使用（推荐）

打开心情记录 App

打开后点页面顶部「保存到手机」，或 Safari 分享 → 添加到主屏幕

方式二：下载 HTML 文件

下载 index.html
  1. 点上面按钮下载
  2. 打开 iPhone「文件」App → 找到下载的文件
  3. 点文件用 Safari 打开
  4. Safari 分享 → 添加到主屏幕

iPhone 注意

  • 不要用无痕浏览（数据会丢）
  • 首次打开需设置男生/女生密码
  • 链接约 60 分钟内有效，请尽快保存到手机
